#### Where to obtain data for this project
The data was obtained from this paper:
Sharma K, Schmitt S, Bergner CG, Tyanova S, Kannaiyan N, Manrique-Hoyos N, Kongi K, Cantuti L, Hanisch UK, Philips MA, Rossner MJ, Mann M, Simons M. 
Cell type- and brain region-resolved mouse brain proteome. Nat Neurosci. 2015 Dec;18(12):1819-31. doi: 10.1038/nn.4160. Epub 2015 Nov 2. PMID: 26523646; PMCID: PMC7116867.

Specifically, supplementary tables 2, 3, 5 and 17.
Downloading the data and putting it in the directory data/_raw/ is enough to run the whole project.
